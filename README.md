A single-file, immersive personal portfolio website featuring a neon-noir aesthetic, integrated audio player, and smooth scroll animations. Designed for Digital Venture Owners and Developers.

## üöÄ Features

* **Single File Architecture:** All HTML, CSS, and JavaScript are contained in one `index.html` file for easy deployment.
* **Cyberpunk UI:** Custom neon glow effects, breathing background animations, and glassmorphism.
* **Interactive Elements:**
    * **Music Player:** Floating play/pause button with pulse animation (Autoplay attempt included).
    * **Scroll Progress:** A visual progress bar at the top of the viewport.
    * **Scroll Reveals:** Elements fade and slide in as you scroll down.
    * **Lightbox:** Click on project images or the "About" image to view them in full screen.
* **Fully Responsive:** Includes a mobile-friendly hamburger menu and adaptive grid layouts.
* **Contact Form:** Integrated with FormSubmit for serverless email handling.

Since this is a static, single-file site, no installation (`npm install`) is required.

1.  **Clone or Download** this repository.
2.  **Assets:** Ensure you have the following files in the same root folder for the site to load correctly:
    * `pfp.jpg` (Profile Picture)
    * `fullme.jpg` (About Section Image)
    * `music.m4a` (Background Audio)
3.  **Run:** simply double-click `index.html` to open it in your browser.
